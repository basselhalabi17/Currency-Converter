# Currency-Converter-using-ASP.NET

This is an currency converter web application implemented using ASP.NET utilizing concepts like MVC, Entity Framework and LINQ.
It has 12 APIs that could be tested via swagger which are:
1. Create Currency
2. Update Currency 
3. Delete Currency
4. Get All Currencies
5. Get Currency By Name
6. Get Highest N Currencies
7. Get Lowest N Currencies
8. Get Most Improved Currencies Between 2 Dates
9. Get Least Improved Currencies Between 2 Dates
10. Convert from one currency to another
11. Login
12. Register
